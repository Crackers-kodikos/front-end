================================================================================
                           VALIDATION RULES
================================================================================

Available Validation Rules (9 pre-built):

  1. required(message)        - Field must not be empty
  2. minLength(min, message)  - Minimum character length
  3. maxLength(max, message)  - Maximum character length
  4. email(message)           - Valid email format
  5. phone(message)           - Valid phone number (10+ digits)
  6. number(message)          - Valid numeric value
  7. minValue(min, message)   - Minimum numeric value
  8. maxValue(max, message)   - Maximum numeric value
  9. pattern(regex, message)  - Custom regex pattern

Each rule can be composed to create complex validation:

  Example:
    name: [
      validationRules.required('Name is required'),
      validationRules.minLength(2, 'Name too short'),
      validationRules.maxLength(100, 'Name too long')
    ]

================================================================================
                             HOW IT WORKS
================================================================================

VALIDATION FLOW:

  User Types in Field
       ↓
  onChange Event → If error exists, clear it
       ↓
  User Leaves Field (Blur)
       ↓
  onBlur Event → Validate field
       ↓
  Show/Hide Error Message
       ↓
  User Submits Form
       ↓
  Validate ALL fields
       ↓
  If Valid: Call onSubmit handler
  If Invalid: Show all errors

ERROR DISPLAY:

  • Red border on field with error
  • Red error message below field
  • Only shown for touched fields
  • Clears automatically as user types

SUCCESS FEEDBACK:

  • Green success message banner
  • Auto-dismisses after 1.5 seconds
  • Navigates to detail page

LOADING STATES:

  • Submit button disabled during submission
  • "Saving..." text during submission
  • Prevents double submissions

BASIC FORM WITH VALIDATION:

  import { useFormValidation, validationRules } from '../hooks/useFormValidation'
  import FormInput from '../component/FormInput'

  export default function MyForm() {
    const form = useFormValidation(
      { name: '', email: '' },
      async (values) => {
        console.log('Valid form submitted:', values)
      },
      {
        name: [
          validationRules.required('Name required'),
          validationRules.minLength(2)
        ],
        email: [
          validationRules.required('Email required'),
          validationRules.email('Invalid email')
        ]
      }
    )

    return (
      <form onSubmit={form.handleSubmit}>
        <FormInput
          label="Name"
          name="name"
          value={form.values.name}
          error={form.errors.name}
          touched={form.touched.name}
          onChange={form.handleChange}
          onBlur={form.handleBlur}
          required
        />
        <FormInput
          label="Email"
          name="email"
          type="email"
          value={form.values.email}
          error={form.errors.email}
          touched={form.touched.email}
          onChange={form.handleChange}
          onBlur={form.handleBlur}
          required
        />
        <button type="submit" disabled={form.isSubmitting}>
          {form.isSubmitting ? 'Submitting...' : 'Submit'}
        </button>
      </form>
    )
  }

DELETE WITH CONFIRMATION:

  import DeleteConfirmModal from '../component/DeleteConfirmModal'

  export default function ItemCard({ item }) {
    const [confirmOpen, setConfirmOpen] = useState(false)
    const [isDeleting, setIsDeleting] = useState(false)

    const handleDelete = async () => {
      setIsDeleting(true)
      try {
        await deleteAPI(item.id)
        // Refresh or navigate
      } finally {
        setIsDeleting(false)
        setConfirmOpen(false)
      }
    }

    return (
      <>
        <button onClick={() => setConfirmOpen(true)}>Delete</button>
        <DeleteConfirmModal
          open={confirmOpen}
          itemName={item.name}
          onCancel={() => setConfirmOpen(false)}
          onConfirm={handleDelete}
          isLoading={isDeleting}
        />
      </>
    )
  }
